# Laravel To Do
To Do List made using [Laravel](http://laravel.com/docs).

## Requirement
- PHP 7.1+
- MySQL 5.7

## How to run
- Clone this project
- Run `composer install`
- Run `php artisan migrate`
- Run `php artisan db:seed`
- Run `php artisan serve`
- Go to http://localhost:8000/
